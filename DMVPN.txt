ESCENARIO 1 — DMVPN Fase 2, IKEv1, EIGRP

Característica clave de Fase 2: el Hub NO hace next-hop-self ni aplica split-horizon en el túnel, para que las rutas EIGRP conserven el next-hop original del spoke origen. Así, cuando un spoke necesita enviar tráfico a otro spoke, EIGRP le indica "el next-hop es el otro spoke", lo que dispara una resolución NHRP y se construye un túnel dinámico spoke-a-spoke.

IOU3 (HUB) — Fase 2

IOU3(config)# interface e0/1
IOU3(config-if)# description WAN hacia ISP
IOU3(config-if)# ip address 200.0.1.2 255.255.255.252
IOU3(config-if)# no shutdown
IOU3(config-if)# exit

IOU3(config)# interface e0/2
IOU3(config-if)# description LAN central
IOU3(config-if)# ip address 192.168.100.1 255.255.255.0
IOU3(config-if)# no shutdown
IOU3(config-if)# exit

IOU3(config)# ip route 0.0.0.0 0.0.0.0 200.0.1.1

! IKEv1 (ISAKMP) — Fase 1
IOU3(config)# crypto isakmp policy 10
IOU3(config-isakmp)# encr aes 256
IOU3(config-isakmp)# hash sha256
IOU3(config-isakmp)# authentication pre-share
IOU3(config-isakmp)# group 14
IOU3(config-isakmp)# lifetime 86400
IOU3(config-isakmp)# exit
IOU3(config)# crypto isakmp key DMVPN_K3Y address 0.0.0.0 0.0.0.0

! Transform-set e IPsec profile
IOU3(config)# crypto ipsec transform-set TS_DMVPN esp-aes 256 esp-sha256-hmac
IOU3(cfg-crypto-trans)# mode transport
IOU3(cfg-crypto-trans)# exit
IOU3(config)# crypto ipsec profile DMVPN-PROFILE
IOU3(ipsec-profile)# set transform-set TS_DMVPN
IOU3(ipsec-profile)# exit

! Tunnel0 — mGRE
IOU3(config)# interface Tunnel0
IOU3(config-if)# description DMVPN Fase 2 - HUB
IOU3(config-if)# ip address 172.16.0.1 255.255.255.0
IOU3(config-if)# no ip redirects
IOU3(config-if)# ip mtu 1400
IOU3(config-if)# ip nhrp authentication DMVPN
IOU3(config-if)# ip nhrp map multicast dynamic
IOU3(config-if)# ip nhrp network-id 100
IOU3(config-if)# ip nhrp holdtime 300
IOU3(config-if)# no ip split-horizon eigrp 100
IOU3(config-if)# no ip next-hop-self eigrp 100
IOU3(config-if)# tunnel source e0/1
IOU3(config-if)# tunnel mode gre multipoint
IOU3(config-if)# tunnel key 100
IOU3(config-if)# tunnel protection ipsec profile DMVPN-PROFILE
IOU3(config-if)# no shutdown
IOU3(config-if)# exit

! EIGRP
IOU3(config)# router eigrp 100
IOU3(config-router)# network 172.16.0.0 0.0.0.255
IOU3(config-router)# network 192.168.100.0 0.0.0.255
IOU3(config-router)# no auto-summary
IOU3(config-router)# exit

! IKEv2

IOU3(config-ikev2-proposal)# integrity sha256
IOU3(config-ikev2-proposal)# group 14
IOU3(config-ikev2-proposal)# exit
IOU3(config)# crypto ikev2 policy DMVPN-POL
IOU3(config-ikev2-policy)# proposal DMVPN-PROP
IOU3(config-ikev2-policy)# exit
IOU3(config)# crypto ikev2 keyring DMVPN-KEYRING
IOU3(config-ikev2-keyring)# peer ANY
IOU3(config-ikev2-keyring-peer)# address 0.0.0.0 0.0.0.0
IOU3(config-ikev2-keyring-peer)# pre-shared-key DMVPN_K3Y
IOU3(config-ikev2-keyring-peer)# exit
IOU3(config-ikev2-keyring)# exit
IOU3(config)# crypto ikev2 profile DMVPN-IKEV2-PROFILE
IOU3(config-ikev2-profile)# match identity remote address 0.0.0.0
IOU3(config-ikev2-profile)# authentication remote pre-share
IOU3(config-ikev2-profile)# authentication local pre-share
IOU3(config-ikev2-profile)# keyring local DMVPN-KEYRING
IOU3(config-ikev2-profile)# exit

! Transform-set e IPsec profile (referencia al perfil IKEv2)
IOU3(config)# crypto ipsec transform-set TS_DMVPN_V2 esp-aes 256 esp-sha256-hmac

IOU3(cfg-crypto-trans)# exit
IOU3(config)# crypto ipsec profile DMVPN-PROFILE-V2
IOU3(ipsec-profile)# set transform-set TS_DMVPN_V2
IOU3(ipsec-profile)# set ikev2-profile DMVPN-IKEV2-PROFILE
IOU3(ipsec-profile)# exit
